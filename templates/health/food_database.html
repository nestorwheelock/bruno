{% trans "Food Preparation Guidelines" %}

  • {% trans "Chicken/Turkey" %}: {% trans "Boil boneless parts until fully cooked. Save broth (no salt) as appetite enhancer. Remove skin if dog has pancreatitis history." %}
  • {% trans "Ground Beef/Lamb" %}: {% trans "Pan-cook until no pink remains. Drain excess fat for leaner dogs, keep fat for underweight dogs." %}
  • {% trans "Eggs" %}: {% trans "Scramble without oil/butter, or hard-boil. Limit 2-3 per day max. Save shells for calcium." %}
  • {% trans "Sardines/Salmon" %}: {% trans "Use water-packed canned without added salt. Fresh fish should be fully cooked." %}
  • {% trans "Liver" %}: {% trans "Boil or pan-cook. Limit to 1-2 servings/week due to vitamin A content." %}

{% trans "Homemade Supplement Preparation" %}

  • {% trans "Eggshell Calcium" %}: {% trans "Save eggshells, boil 10 minutes, dry completely, grind to fine powder. 1/2 tsp = ~400mg calcium. Store in airtight container." %}
  • {% trans "Fish Oil Dosing" %}: {% trans "Target 50-100mg EPA+DHA per kg body weight. Check capsule label for EPA/DHA content (not just 'fish oil')." %}
  • {% trans "Multivitamin" %}: {% trans "Use canine-specific multivitamin daily to cover micronutrient gaps from home-cooked diet." %}
