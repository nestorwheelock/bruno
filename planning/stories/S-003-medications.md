# S-003: Medication Tracking

**Story Type**: User Story
**Priority**: High
**Status**: PENDING

## User Story
**As a** caregiver
**I want to** track Bruno's medications and record when doses are given
**So that** I ensure he gets his medicine on schedule

## Acceptance Criteria
- [ ] Can add medications with name, dosage, frequency
- [ ] Can record when a dose is given
- [ ] Can see today's doses
- [ ] Multiple users can record doses

## Definition of Done
- [ ] Medication model
- [ ] MedicationDose model
- [ ] Views for CRUD operations
- [ ] Tests for models and views
- [ ] >95% test coverage
